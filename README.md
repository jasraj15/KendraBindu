🧠 Kendrabindu
AI-Powered Focus & Digital Discipline System

Kendrabindu is a unified focus ecosystem combining a Chrome Extension and Android app designed to reduce digital distractions and rebuild attention spans.

Instead of blindly blocking apps, Kendrabindu uses smart conditional rules, keyword filtering, and mindful nudges to promote deep work and intentional digital usage.

🚨 Problem

Students and professionals lose an average of 4+ hours daily due to:

Digital overstimulation

Endless scrolling

Chronic multitasking

Fragmented attention

Existing productivity tools block everything — but they don’t fix focus behavior.

💡 Our Solution

Kendrabindu intelligently filters distractions while preserving productive usage.

Example:
✔ Allows YouTube coding tutorials
❌ Blocks YouTube entertainment & gaming

It combines:

Smart URL & Keyword Blocking

Focus Mode (Allowlist / Blocklist)

Screen Time Tracking

Mindful Reminders

Distraction Analytics
